
#### Commands

 In MSFconsole
 
Display which workspace is currently active
```
workspace
```

Shows previous targets
```
hosts
```

Create workspace 
```
workspace -a <workspace_name>
```

Switch workspace
```
workspace <workspace_name>
```

Delete workspace
```
workspace -d <workspace_name>
```

Rename workspace
```
workspace -r <current_workspace_name> <new_workspace_name>
```
